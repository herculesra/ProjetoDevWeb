# Front do GiftLoucos

## Executando o projeto

* Dentro da pasta raiz, execute:

```
$ npm install
```

* Para inicializar o projeto:

```
$ npm start
```

**OBS**: É importante que o Backend esteja em execução, caso contrário, você receberá um erro de comunicação.
